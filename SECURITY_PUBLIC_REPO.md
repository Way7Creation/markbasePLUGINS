## Этот репозиторий публичный на GitHub

**Никогда не добавляйте сюда:**

- `.env` и любые файлы с переменными окружения
- API-ключи, токены, пароли
- Приватные ключи, сертификаты
- IP-адреса серверов, внутренние домены
- Данные пользователей, логи с персональной информацией
- Внутреннюю коммерческую информацию

## Что можно коммитить

- Публичную документацию для разработчиков
- Примеры кода (без реальных ключей)
- Спецификации API (plugin.json, README)
- `.env.example` — только шаблоны без значений

## Перед коммитом

Убедитесь, что в `.gitignore` исключены все чувствительные файлы.  
При сомнениях — **не коммитить**.
